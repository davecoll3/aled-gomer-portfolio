# Aled Gomer Portfolio

A simple, responsive portfolio site for Aled Gomer (actor, writer, theatre maker, and facilitator). The site presents an about section, project showcase videos, facilitating work, and contact links.

## Live site
- Domain configured via `CNAME` for GitHub Pages.

## Features
- Responsive single‑page layout with anchored navigation.
- Project highlights with embedded YouTube videos.
- Facilitating section with image gallery.
- Contact links (Spotlight, Instagram, Email).

## Tech stack
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome

## Project structure
- `index.html` — main page markup.
- `assets/` — images, CSS, and JavaScript.
- `site.webmanifest` — PWA/manifest metadata.
- `CNAME` — custom domain for GitHub Pages.

## Local development
This is a static site. You can open `index.html` directly in a browser, or serve the folder with any static file server.

Example (Python):
```
python3 -m http.server
```
